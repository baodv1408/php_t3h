Thuộc tính style

Thuộc tính style áp dụng cho mọi thẻ html

Thuộc tính style sẽ chỉ ra cách hiển thị cho 1 thẻ html như màu sắc , kích cỡ , font ...

Thuộc tính style có giá trị là 1 cặp key và value theo định dạng key:value

Nếu 1 thẻ có nhiều cặp key và value trong style chúng sẽ được phân cách bằng dấu ;

key1:value1;key2:value2;key3:value3 ...

Để sử dụng tốt thuộc tính style các bạn cần có kiến thức về css
